# PNGReadWrite
 PNG file IO - Deep color compatible

 [PNG Suite](http://www.schaik.com/pngsuite/) passed

## Requirement
 .NET 6 - windows

## Install
[Download DLL](https://github.com/tk-yoshimura/PNGReadWrite/releases)

- Nuget include: System.Drawing.Common
- Enable WPF (*.csproj)
```xml
<PropertyGroup>
    <UseWPF>true</UseWPF>
</PropertyGroup>
```
